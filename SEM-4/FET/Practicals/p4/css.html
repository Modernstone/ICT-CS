CSS Properties and Their Usage

color

Sets the text color.

color: red;

background-color

Sets the background color.

background-color: blue;

font-size

Defines the text size.

font-size: 16px;

margin

Defines space outside an element.

margin: 10px;

padding

Defines space inside an element.

padding: 20px;

border

Defines the border style.

border: 2px solid black;

width & height

Sets element dimensions.

width: 100px; height: 50px;

display

Defines display behavior.

display: flex;

position

Positions elements.

position: absolute;

box-shadow

Adds shadow to elements.

box-shadow: 5px 5px 10px gray;
